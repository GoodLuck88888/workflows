Anita modeler is odoo's code generator and data relation designer. It is part of our low-code system (modeler, workflow, form designer, reporter). It can collect data from odoo and generate code for odoo. It is two-way. It includes a server and a client. The client is used to generate code for odoo. The server is used to manage data. They are all written in python. it can be used in windows, linux, mac.
